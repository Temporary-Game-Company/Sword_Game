Detailed Description

SpawnAI component. Provides the base methods required to interact with the spawner.

This component can be used out of the box on existing GameObjects that get spawned.
This class interacts with the Spawner and handles all interaction between the two. It also makes use of InstanceManager in order to work with Pool Manager by Path-o-Logical (if it is installed).
Simply interact with this class in your killing method and call Remove() on this component last in the method, you will not need to call Destroy(gameObject) in order to destroy the GameObject as this class handles all of that.
