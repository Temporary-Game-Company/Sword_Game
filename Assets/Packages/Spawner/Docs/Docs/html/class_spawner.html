Detailed Description

Spawns prefabs, either in waves, at once or continually till all enemies are spawned.

Controls the spawning of selected perfabs, useful for making enemy spawn points.
It supports a variety of spawn modes, which allows you to bend the system to fit your needs.
This class is required for the system to work, you will need to place this class on a GameObject with a tag of Spawner (However, this is changeable within the SpawnAI class).
