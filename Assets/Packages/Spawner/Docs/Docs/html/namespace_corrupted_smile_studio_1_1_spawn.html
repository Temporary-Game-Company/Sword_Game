enum CorruptedSmileStudio.Spawn.SpawnModes

The different spawn modes supported.

Enumerator
Normal 

This will continually spawn units until the limit, then spawn whenever a unit dies. Use this mode with a trigger that disables the spawner.

Once 

This will spawn units till the limit is hit then stop.

Wave 

This will spawn units in waves. Once all units are dead it will spawn a new wave.

TimedWave 

This will spawn units in waves, once the specified time has passed a new wave will spawn regardless of whether there was still units left.

TimeSplitWave 

This will spawn units in waves, once all units are dead it will after the specified time spawn a new wave.
